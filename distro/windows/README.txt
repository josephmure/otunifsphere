# otunifsphere module

Examples
  Template examples can be found here: C:\Python27\Lib\site-packages\otunifsphere\examples\otunifsphere

Installation
  Silent install can be done with the switch /S (e.g. openturns-otunifsphere-0.0-py27.exe /S).
  To test otunifsphere module installation, in an OpenTURNS python console type:
    import otunifsphere
    print otunifsphere.__version__

Uninstallation
  otunifsphere module will be automatically uninstalled at OpenTURNS removal.
  otunifsphere module can be removed from OpenTURNS with uninst-template.exe uninstaller.
  Silent uninstall can be done with the switch /S.
